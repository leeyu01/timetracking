#Time Tracking

DCAA-compliant timekeeping & cost accounting system

## Quick start

- Node/NPM or pnpm/Yarn version: e.g. Node >= 18
- `npm install`
- `npm run dev` (backend)
- `cd frontend && npm install && npm run dev`

## Architecture
- Backend: Node.js (NestJS / TypeScript)
- Frontend: React + TypeScript
- DB: PostgreSQL
- Auth: Keycloak (OIDC)
- CI: GitHub Actions
- Deployment: Docker + Nginx
